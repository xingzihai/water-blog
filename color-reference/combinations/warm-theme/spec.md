# Warm Theme · 温暖主题配色方案

> 以灰粉为主调，营造温暖、亲和、怀旧的氛围

---

## 核心配置

```css
:root {
  /* 主色调 */
  --theme-primary: var(--pink-200);      /* #D4C4BC */
  --theme-primary-light: var(--pink-100); /* #EAD9D4 */
  --theme-primary-dark: var(--pink-400);  /* #B48C84 */
  
  /* 辅助色 */
  --theme-accent: var(--orange-200);      /* #D8C8A8 */
  
  /* 背景 */
  --theme-bg: var(--bg-300);              /* #E8DFD6 */
  --theme-bg-card: var(--bg-200);         /* #F5F1ED */
  
  /* 文字 */
  --theme-text: var(--text-200);          /* #5C4A3D */
  --theme-text-muted: var(--text-400);    /* #9A9A9A */
}
```

---

## 色彩分布

| 用途 | 颜色 | HEX |
|------|------|-----|
| 页面背景 | 暖米 | `#E8DFD6` |
| 卡片背景 | 米白 | `#F5F1ED` |
| 强调区块 | 灰粉 | `#D4C4BC` |
| 标题文字 | 深褐 | `#5C4A3D` |
| 正文文字 | 烟灰 | `#7A7A7A` |
| 交互元素 | 灰橙 | `#D8C8A8` |
| 边框装饰 | 雾霾蓝 | `#A8B4B8` |

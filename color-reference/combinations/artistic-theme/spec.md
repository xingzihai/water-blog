# Artistic Theme · 艺术主题配色方案

> 以灰紫为主调，营造优雅、神秘、艺术的氛围

---

## 核心配置

```css
:root {
  /* 主色调 */
  --theme-primary: var(--purple-200);     /* #B8A8B8 */
  --theme-primary-light: var(--purple-100); /* #D8D0D8 */
  --theme-primary-dark: var(--purple-400);  /* #786878 */
  
  /* 辅助色 */
  --theme-accent: var(--rose-200);        /* #D8B0C0 */
  
  /* 背景 */
  --theme-bg: var(--bg-200);              /* #F5F1ED */
  --theme-bg-card: var(--purple-100);     /* #D8D0D8 */
  
  /* 文字 */
  --theme-text: var(--text-200);          /* #5C4A3D */
  --theme-text-muted: var(--text-400);    /* #9A9A9A */
}
```

---

## 色彩分布

| 用途 | 颜色 | HEX |
|------|------|-----|
| 页面背景 | 米白 | `#F5F1ED` |
| 卡片背景 | 浅灰紫 | `#D8D0D8` |
| 强调区块 | 灰紫 | `#B8A8B8` |
| 标题文字 | 深褐 | `#5C4A3D` |
| 正文文字 | 烟灰 | `#7A7A7A` |
| 装饰元素 | 灰玫红 | `#D8B0C0` |
| 深色强调 | 最深灰紫 | `#584858` |

# Cool Theme · 冷静主题配色方案

> 以雾霾蓝为主调，营造专业、理性、可信的氛围

---

## 核心配置

```css
:root {
  /* 主色调 */
  --theme-primary: var(--blue-200);      /* #A8B4B8 */
  --theme-primary-light: var(--blue-100); /* #C4CED2 */
  --theme-primary-dark: var(--blue-400);  /* #6A7A82 */
  
  /* 辅助色 */
  --theme-accent: var(--green-200);       /* #9CAA9C */
  
  /* 背景 */
  --theme-bg: var(--bg-100);              /* #FAF8F5 */
  --theme-bg-card: var(--bg-200);         /* #F5F1ED */
  
  /* 文字 */
  --theme-text: var(--text-200);          /* #5C4A3D */
  --theme-text-muted: var(--text-400);    /* #9A9A9A */
}
```

---

## 色彩分布

| 用途 | 颜色 | HEX |
|------|------|-----|
| 页面背景 | 最浅米 | `#FAF8F5` |
| 卡片背景 | 米白 | `#F5F1ED` |
| 导航/边框 | 雾霾蓝 | `#A8B4B8` |
| 标题文字 | 深褐 | `#5C4A3D` |
| 正文文字 | 烟灰 | `#7A7A7A` |
| 成功状态 | 灰绿 | `#9CAA9C` |
| 深色区块 | 深雾霾蓝 | `#4A5A62` |

# Neutral Theme · 中性主题配色方案

> 以米色/灰色为主调，营造简洁、现代、通用的氛围

---

## 核心配置

```css
:root {
  /* 主色调 */
  --theme-primary: var(--bg-300);         /* #E8DFD6 */
  --theme-primary-light: var(--bg-200);   /* #F5F1ED */
  --theme-primary-dark: var(--bg-500);    /* #C9B8A8 */
  
  /* 辅助色 */
  --theme-accent: var(--blue-200);        /* #A8B4B8 */
  
  /* 背景 */
  --theme-bg: var(--bg-100);              /* #FAF8F5 */
  --theme-bg-card: var(--bg-200);         /* #F5F1ED */
  
  /* 文字 */
  --theme-text: var(--text-200);          /* #5C4A3D */
  --theme-text-muted: var(--text-400);    /* #9A9A9A */
}
```

---

## 色彩分布

| 用途 | 颜色 | HEX |
|------|------|-----|
| 页面背景 | 最浅米 | `#FAF8F5` |
| 卡片背景 | 米白 | `#F5F1ED` |
| 区块背景 | 暖米 | `#E8DFD6` |
| 标题文字 | 深褐 | `#5C4A3D` |
| 正文文字 | 烟灰 | `#7A7A7A` |
| 边框/装饰 | 雾霾蓝 | `#A8B4B8` |
| 强调区块 | 最深米 | `#C9B8A8` |

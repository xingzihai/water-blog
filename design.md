# 美学设计规范 - Aesthetic Portfolio

> 从心理学与美学角度构建的视觉系统

---

## 一、心理学基础设计原则

### 1.1 认知负荷理论 (Cognitive Load Theory)

**原则**：人脑工作记忆容量有限（4±1 个元素），过度信息会导致认知疲劳。

**应用**：
- **每屏焦点 ≤ 3 个**：用户注意力有限，一屏只传达一个核心信息
- **渐进式揭示**：信息分层次呈现，避免一次性轰炸
- **留白 > 40%**：视觉呼吸空间降低认知压力
- **分组原则**：相关元素物理聚合，减少眼球移动距离

### 1.2 格式塔心理学 (Gestalt Principles)

**相近性 (Proximity)**：
- 相关内容物理靠近，形成视觉单元
- 卡片内部间距 < 卡片之间间距

**相似性 (Similarity)**：
- 相同功能的按钮使用相同视觉样式
- 建立视觉一致性后，用户可预测交互方式

**连续性 (Continuity)**：
- 视觉引导线（如装饰线条）引导眼球移动
- 阅读流：Z 型（宽屏）或 F 型（移动端）

**闭合性 (Closure)**：
- 不完整图形由大脑自动补全
- 用于创造趣味性和参与感

### 1.3 色彩心理学 (Color Psychology)

**莫兰迪色系的心理效应**：

| 色相 | 心理效应 | 应用场景 |
|------|----------|----------|
| 灰粉 #D4C4BC | 温暖、安全、怀旧 | 背景、卡片、次要元素 |
| 雾霾蓝 #A8B4B8 | 冷静、信任、思考 | 边框、装饰、交互反馈 |
| 灰绿 #9CAA9C | 自然、平和、治愈 | 成功状态、点缀 |
| 暖米 #E8DFD6 | 温和、包容、舒适 | 主背景 |
| 烟灰 #7A7A7A | 稳重、专业、中立 | 正文文字 |
| 深褐 #5C4A3D | 沉稳、历史、品质 | 标题、重点强调 |

**低饱和度的作用**：
- 降低视觉刺激，适合长时间阅读
- 营造"慢体验"，引导用户放慢节奏
- 减少情绪波动，保持平静专注

### 1.4 排版心理学 (Typography Psychology)

**衬线字体 (Serif)**：
- 心理感受：传统、优雅、可信、有历史感
- 适用：标题、引用、强调内容
- 代表：Playfair Display, Cormorant, Crimson Text

**无衬线字体 (Sans-serif)**：
- 心理感受：现代、简洁、友好、易读
- 适用：正文、UI 元素、导航
- 代表：Inter, Nunito, Source Sans Pro

**字体配对原则**：
- 标题用衬线，正文用无衬线 → 对比产生层次
- 字重对比：标题 Bold (700)，正文 Regular (400)
- 字号对比：标题 ≥ 正文 × 2

### 2.2 字体系统

**字体家族**：
```css
:root {
  --font-display: 'Cormorant Garamond', 'Playfair Display', Georgia, serif;
  --font-body: 'Inter', 'Nunito', -apple-system, sans-serif;
  --font-mono: 'JetBrains Mono', 'Fira Code', monospace;
}
```

**字体规格**：
```
标题 H1     48px / 700 / 1.2行高 / --font-display
副标题 H2   32px / 600 / 1.3行高 / --font-display
小标题 H3   24px / 500 / 1.4行高 / --font-display
正文        18px / 400 / 1.7行高 / --font-body
辅助文字    14px / 400 / 1.5行高 / --font-body
微文字      12px / 400 / 1.4行高 / --font-mono
```

**心理考量**：
- 标题 48px：冲击感、注意力抓取、建立视觉锚点
- 正文 18px：舒适的阅读尺寸，减少眼睛疲劳
- 行高 1.7：给文字呼吸空间，提升可读性
- 字重 400：不粗不细，温和友好

### 2.3 间距系统

**8px 基准网格**：
```css
:root {
  --space-xs:   8px;   /* 元素内部间距 */
  --space-sm:   16px;  /* 相关元素间距 */
  --space-md:   24px;  /* 模块间距 */
  --space-lg:   48px;  /* 区块间距 */
  --space-xl:   80px;  /* 大区块间距 */
  --space-2xl:  120px; /* 页面级间距 */
}
```

**心理效应**：
- 紧密间距 → 关联性强、信息密度高、专业感
- 宽松间距 → 轻松感、呼吸感、高级感
- 我们选择**偏宽松**：传达舒适、不急躁的体验

### 2.4 圆角与阴影

```css
:root {
  --radius-sm: 4px;   /* 小按钮、标签 */
  --radius-md: 8px;   /* 输入框、小卡片 */
  --radius-lg: 12px;  /* 大卡片 */
  --radius-xl: 20px;  /* 特殊元素 */
  
  --shadow-soft: 0 4px 20px rgba(92, 74, 61, 0.08);
  --shadow-hover: 0 8px 30px rgba(92, 74, 61, 0.12);
  --shadow-card: 0 2px 12px rgba(92, 74, 61, 0.06);
}
```

**心理考量**：
- 小圆角 → 精致、现代、克制
- 大圆角 → 友好、温暖、亲民
- 柔和阴影 → 深度感、层次感、悬浮感

## 三、交互体验设计

### 3.1 微交互原则

**反馈时间**：
- 即时反馈：0-100ms（按钮点击、hover）
- 短过渡：200-300ms（卡片悬浮、展开）
- 长过渡：400-600ms（页面切换、模态框）

**缓动函数**：
```css
/* 自然缓动 - 模拟物理世界的减速 */
--ease-out: cubic-bezier(0.25, 0.46, 0.45, 0.94);

/* 弹性缓动 - 带一点点回弹 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);

/* 平滑缓动 - 标准曲线 */
--ease-smooth: cubic-bezier(0.4, 0, 0.2, 1);
```

**心理效应**：
- 即时反馈 → 系统响应灵敏、掌控感
- 缓动曲线 → 自然感、不机械、有生命力
- 过渡时长 → 太快感觉廉价，太慢感觉迟钝

### 3.2 悬停效果

**卡片悬停**：
```css
.card {
  transition: transform 0.3s var(--ease-out), 
              box-shadow 0.3s var(--ease-out);
}
.card:hover {
  transform: translateY(-4px);
  box-shadow: var(--shadow-hover);
}
```

**心理目的**：
- 上浮 → "我选中了这个" 的掌控感
- 阴影加深 → 物体更近的错觉

### 3.3 滚动体验

**平滑滚动**：
```css
html {
  scroll-behavior: smooth;
}
```

**滚动进度指示**：
- 顶部细线指示阅读进度
- 心理作用：进度感知 → 掌控感 → 降低焦虑

### 3.4 光标与指针

| 元素 | 光标 | 心理暗示 |
|------|------|----------|
| 按钮/链接 | `pointer` | "可点击" |
| 可拖拽 | `grab` / `grabbing` | "可抓取" |
| 输入框 | `text` | "可输入" |
| 禁用 | `not-allowed` | "不可操作" |

## 四、页面结构与视觉流

### 4.1 首页布局 (F型阅读流)

```
┌────────────────────────────────────────┐
│            导航栏 (固定顶部)             │
├────────────────────────────────────────┤
│                                        │
│     ╔══════════════════════════╗       │
│     ║                          ║       │
│     ║    大标题 (视觉锚点)      ║       │  ← 第一视线落点
│     ║    副标题/简介            ║       │
│     ║                          ║       │
│     ╚══════════════════════════╝       │
│                                        │
│  ─────────────────────────────────     │  ← 装饰分隔线
│                                        │
│     ○ 作品卡片    ○ 作品卡片           │  ← 左右对称
│                                        │
│     ○ 作品卡片    ○ 作品卡片           │
│                                        │
├────────────────────────────────────────┤
│              页脚信息                   │
└────────────────────────────────────────┘
```

### 4.2 视觉层级

1. **第一层（吸引）**：大标题、主图
2. **第二层（导航）**：导航栏、CTA 按钮
3. **第三层（内容）**：作品卡片、段落文字
4. **第四层（细节）**：辅助信息、页脚

```css
/* 移动优先 */

--breakpoint-md: 768px;   /* 平板竖屏 */
```

- 标题缩小（48px → 32px）
- 单列布局
- 导航收起为汉堡菜单
- 触控目标 ≥ 44px（Fitts 定律）

## 六、无障碍设计

- **色彩对比度**：正文文字对比度 ≥ 4.5:1
- **字体大小**：支持 200% 放大不破坏布局
- **键盘导航**：所有交互元素可通过键盘访问
- **焦点状态**：清晰的 focus 样式
- **语义化 HTML**：正确使用 heading、main、nav、article

## 八、核心体验目标

### 用户心理旅程

1. **进入** → 感受平静、不急躁
2. **浏览** → 享受阅读、不被打扰
3. **交互** → 反馈即时、感觉流畅
4. **停留** → 愿意慢下来、欣赏细节
5. **离开** → 带着美好印象、愿意回来

### 设计哲学

> **"少即是多，慢即是快"**
> 
> 不追求信息量的堆砌，而追求每一个像素的温度。
> 让用户在这里慢下来，感受设计本身的语言。

## 使用方式

```html
<!-- 在背后世界层引用 -->
<link rel="stylesheet" href="behind-world/color-reference/css-variables.css">
```

```css
.behind-world {
  background: var(--bg-300);  /* 睡莲水面 */
  color: var(--text-100);     /* 深水绿 */
}

.accent {
  color: var(--sunrise-300);  /* 日出核心橙 */
}
```

## 莫奈设计原则

1. **光的变化**：同一场景不同光线下的色彩变化
2. **并列笔触**：纯色并置，让眼睛自行混合
3. **模糊边界**：物体边缘柔和，融入环境
4. **瞬息捕捉**：表现当下的光影瞬间

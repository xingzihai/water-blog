# 光标美学设计规范

> 创建日期：2026-04-02
> 状态：设计阶段

---

## 核心理念

**光标不是指针，是通往另一个维度的窗口。**

---

## 设计层级

### 1. 基础形态

| 属性 | 值 |
|------|-----|
| 形状 | 圆形 |
| 默认尺寸 | 80px 直径 |
| 边缘 | 柔和模糊（4-8px feather） |
| 跟随延迟 | 100-150ms（弹簧阻尼） |

---

## 2. "背后的页面"概念

### 视觉效果

光标像一个**窗口**，显示隐藏在当前页面"背后"的另一个世界。

```
┌─────────────────────────────┐
│                             │
│      当前页面（表层）         │
│                             │
│         ┌───────┐           │
│         │       │           │
│         │ 背后  │ ← 光标窗口 │
│         │ 页面  │           │
│         │       │           │
│         └───────┘           │
│                             │
└─────────────────────────────┘
```

### "背后的页面"设计选项

| 方案 | 说明 | 效果 |
|------|------|------|
| **A. 负片** | 色相反转（明↔暗、暖↔冷） | 神秘、对立 |
| **B. 平行世界** | 相同布局，不同配色 | 和谐、对比 |
| **C. 时间镜像** | 现在的自己 vs 过去的自己 | 叙事感 |
| **D. 维度穿越** | 印象派 ↔ 赛博朋克/极简 | 冲突美学 |

### 技术实现

```css
/* 方案：clip-path + 背后层 */

/* 背后页面层 */
.behind-world {
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9998;
  /* 变换效果 */
  filter: invert(0.8) hue-rotate(180deg);
  /* 或完全不同的设计 */
}

/* 光标窗口 */
.cursor-portal {
  position: fixed;
  width: 80px;
  height: 80px;
  border-radius: 50%;
  z-index: 9999;
  /* 显示背后层 */
  /* 通过 JS 同步位置 */
}
```

## 4. 卡片特殊交互

### "光"卡片 - 太阳光源

**光标形态**：太阳 ☀

**交互效果**：
- 光标变成太阳图形
- 卡片内部产生光照渐变
- 光照范围内元素变亮
- 光芒四射动画

```
┌────────────────────────┐
│                        │
│    卡片内容            │
│         ☀              │ ← 光标 = 太阳
│      ╱  │  ╲           │ ← 光芒
│     ╱   │   ╲          │
│   受光区（变亮）        │
└────────────────────────┘
```

**技术实现**：
```css
.light-card {
  position: relative;
}

.light-card::before {
  content: '';
  position: absolute;
  /* 光照渐变，中心跟随光标 */
  background: radial-gradient(
    circle at var(--cursor-x, 50%) var(--cursor-y, 50%),
    rgba(255, 220, 180, 0.3) 0%,
    transparent 60%
  );
}
```

---

### "影"卡片 - 投影源

**光标形态**：光源点 ●

**交互效果**：
- 光标变成发光点
- 卡片内元素产生投影
- 投影方向跟随光标位置
- 距离越远，投影越长

```
┌────────────────────────┐
│                        │
│    卡片元素            │
│         ●              │ ← 光标 = 光源
│                        │
│      ▓▓▓               │ ← 投影
│       ▓▓▓              │
└────────────────────────┘
```

**技术实现**：
```javascript
// 计算投影方向
const dx = cursorX - elementX;
const dy = cursorY - elementY;
const angle = Math.atan2(dy, dx);
const distance = Math.sqrt(dx*dx + dy*dy);
const shadowLength = distance * 0.3;

element.style.filter = `drop-shadow(${-Math.cos(angle) * shadowLength}px ${-Math.sin(angle) * shadowLength}px 10px rgba(0,0,0,0.3))`;
```

## 5. 状态设计

| 状态 | 外观 | 变化 |
|------|------|------|
| 默认移动 | 圆形窗口，显示背后页面 | 平滑跟随 |
| 快速移动 | 窗口拉长变形 | 速度反馈 |
| 悬停链接 | 窗口缩小，显示链接预览 | 即时反馈 |
| 点击 | 脉冲涟漪 | 确认反馈 |
| 文字选择 | 变成细线（I-beam） | 形态变化 |
| 等待加载 | 转圈动画 | 状态提示 |

## 6. 技术架构

### DOM 结构

```html
<!-- 光标层 -->
<div class="cursor-layer">
  <div class="cursor-portal">
    <!-- 背后页面内容（通过 CSS 变换显示） -->
  </div>
  <div class="cursor-glow"></div>
  <div class="cursor-effects"></div>
</div>

<!-- 背后世界层 -->
<div class="behind-world">
  <!-- 平行世界的页面内容 -->
</div>
```

### JS 模块

```javascript
// main.js 新增模块
initCustomCursor() {
  // 1. 创建光标 DOM
  // 2. 监听 mousemove 更新位置
  // 3. 检测当前 section 更新颜色
  // 4. 检测当前卡片更新交互模式
  // 5. 处理点击/悬停状态
}
```

### 性能优化

- 使用 `requestAnimationFrame` 更新位置
- 使用 CSS `transform` 而非 `left/top`
- 使用 `will-change: transform` 启用 GPU 加速
- 隐藏默认光标：`body { cursor: none; }`

# 动态主头像设计调研任务

## 项目背景

**项目名称**：xingzihai 个人网站（冰水泡枸杞）

**设计理念**：
- 印象派 × 莫兰迪融合美学
- 莫奈光影 + 莫兰迪沉静
- 页面背景使用莫兰迪灰调，主头像使用马卡龙配色作为视觉焦点

**技术约束**：
- 纯 CSS 3D Transform + JavaScript 动画
- 无外部 3D 库依赖
- 鼠标跟随交互（全页面响应）
- 性能友好（GPU 加速）

**当前问题**：
- 六棱柱几何感太生硬
- 马卡龙配色与整体氛围不够协调
- 缺乏艺术感和识别度

**设计目标**：
- 高质量的艺术化动态主头像
- 与印象派美学契合
- 有识别度和记忆点
- 交互体验舒适自然

**色彩系统**：
- 页面背景：莫兰迪灰调（#E8DFD6 暖米、#F5F1ED 米白）
- 强调色：莫奈光影（#FFB088 日出橙、#A8B4B8 雾霾蓝、#D8B0C0 睡莲粉）
- 主头像：可使用马卡龙色或与整体协调的其他方案

**意象关键词**：
- 冰水泡枸杞 → 冷与暖的融合、渗透、时间流逝
- 星光落入冰水 → 光、透明、折射
- 印象派 → 模糊、光斑、瞬时性、自然

---

## 调研任务要求

请调研你负责的风格方向，给出 **两个** 具体的动态主头像设计方案。

每个方案需包含：
1. **设计概念**：核心理念和视觉隐喻
2. **形态描述**：具体形状、结构、层次
3. **配色方案**：具体色值（HEX）和应用方式
4. **动态效果**：静止时和交互时的动画
5. **技术实现**：CSS/JS 实现思路
6. **与整体协调性**：如何与印象派莫兰迪风格融合

**禁止**：
- 直接复制现有设计
- 使用外部 3D 库
- 过于复杂导致性能问题

**鼓励**：
- 创新性的几何组合
- 光影与材质的创意表现
- 有故事性的设计概念
